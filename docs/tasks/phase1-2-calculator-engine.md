# Phase 1.2: Calculator Engine Implementation

## 작업 배경
수학적 수식을 실제로 계산하고 결과를 반환하는 핵심 엔진입니다. `math.js`를 활용하여 기본 연산부터 삼각함수, 로그, 상수 계산까지 처리하며, 사용자에게 친절한 에러 메시지와 포맷팅된 결과를 제공합니다.
