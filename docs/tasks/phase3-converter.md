# Phase 3: Unit Converter Implementation

## 작업 배경
단순 계산을 넘어 길이, 무게, 온도 등 다양한 물리량을 변환할 수 있는 기능을 제공합니다. 코어 변환 로직은 TDD로 구현하여 정확성을 보장하고, UI는 기존 계산기 UI와 조화를 이루도록 설계합니다.

## 작업 내용 (TDD + Manual Testing)
- [ ] **3.1** Converter Engine 구현 (TDD)
  - 길이, 무게, 온도 변환 공식 구현
  - 단위 카테고리 관리 시스템
- [ ] **3.2** Converter UI 구현 (Manual Testing)
  - 입력/출력 필드 및 단위 선택 드롭다운
  - 카테고리 전환 및 단위 스왑 기능

## 인수 조건
- [ ] 모든 단위 변환이 표준 공식에 따라 정확하게 계산됨 (커버리지 90%+)
- [ ] 사용자가 입력하는 즉시 결과가 실시간으로 변환되어 표시됨
- [ ] 카테고리별 단위 목록이 정확하게 필터링되어 노출됨
- [ ] UI 레이아웃이 계산기 모드와 일관성을 유지함
