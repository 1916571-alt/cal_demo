# Phase 1.4: Utility Functions Implementation

## 작업 배경
앱 전반에서 재사용되는 포맷팅, 유효성 검사, 성능 최적화 도구들을 구현합니다. 코드의 중복을 줄이고 품질을 높이는 독립적인 유틸리티들입니다.

## 작업 내용 (TDD Tasks)
- [ ] **1.4.1** 숫자 포맷터 테스트 및 구현 (천 단위 컴마 등)
- [ ] **1.4.2** 수식 포맷터 테스트 및 구현 (공백 처리 등)
- [ ] **1.4.3** 입력 값 유효성 검사기 테스트 및 구현
- [ ] **1.4.4** 디바운스(Debounce), 메모이제이션(Memoization) 테스트 및 구현

## 인수 조건
- [ ] 숫자가 사용자 읽기 편한 포맷으로 정확하게 변환됨
- [ ] 수식 입력 시 실시간으로 가독성 있게 포맷팅됨
- [ ] 유효하지 않은 입력(연산자 중복 등)이 즉각 필터링됨
- [ ] 성능 관련 유틸리티들이 의도한 대로 동작함 (중복 계산 방지 등)
- [ ] 코어 로직 테스트 커버리지가 90% 이상을 달성함
