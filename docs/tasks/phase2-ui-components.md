# Phase 2: UI Components & App Integration

## 작업 배경
사용자가 계산기와 상호작용하는 시각적 인터페이스를 구현합니다. PRD와 디자인 파일을 바탕으로 직관적이고 반응이 빠른 UI 컴포넌트들을 제작하고, 코어 엔진과 연결하여 실제 동작하는 애플리케이션으로 통합합니다.

## 작업 내용 (Manual Testing Tasks)
- [ ] **2.1** Display 컴포넌트 구현 (결과창, 히스토리, 에러 표시)
- [ ] **2.2** Keypad 컴포넌트 구현 (숫자, 연산자, 제어 버튼 그리드)
- [ ] **2.3** Function Chips 구현 (스크롤 가능한 고급 함수 버튼)
- [ ] **2.4** Header 컴포넌트 구현 (메뉴, 각도 모드 토글)
- [ ] **2.5** Mode Switch 컴포넌트 구현 (계산기/변환기 탭)
- [ ] **2.6** 메인 앱 통합 및 키보드 지원

## 인수 조건
- [ ] 모든 UI 컴포넌트가 디자인 가이드라인(`design/screen.png`)과 일치함
- [ ] 버튼 클릭 및 키보드 입력이 정상적으로 동작함
- [ ] 결과창이 실시간으로 업데이트되고, 산식 가독성이 확보됨
- [ ] 각도 모드(DEG/RAD) 및 테마 설정이 UI에 즉각 반영됨
- [ ] 모든 UI 요소가 반응형으로 설계되어 다양한 화면 크기에서 정상 노출됨
- [ ] **참고**: UI는 자동화 테스트 대신 수동 테스트로 검증함
