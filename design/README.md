# Design Reference Files

This directory contains the original design reference files for the Engineering Calculator.

## Files

### `screen.png`
Visual mockup/screenshot of the calculator interface showing:
- Dark mode design
- Calculator layout with keypad
- Display area
- Function chips
- Color scheme and typography

### `code.html`
HTML/CSS reference implementation including:
- Complete UI structure
- Tailwind CSS styling
- Component layout
- Design tokens (colors, fonts, spacing)
- Interactive elements

## Usage

These files serve as the design reference for implementing the calculator UI. The actual implementation is in the `src/` directory.

**Note**: These are reference files only and are not part of the build process.
